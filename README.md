# RacketAI 🎾

**Trova la tua racchetta da padel perfetta** — Test AI gratuito con 178 racchette nel database.

🔗 **Live:** https://stinkipadel.github.io

## Funzionalità
- Test veloce (6 domande) e test completo (15 domande)
- Algoritmo AI di matching racchette
- Database 178 racchette con filtri, confronto e prezzi
- Trova la racchetta per il tuo partner di coppia
- Dark mode, Safari iOS ottimizzato
- Nessuna registrazione richiesta

## Come aggiornare
1. Modifica `index.html`
2. Fai commit su GitHub
3. Il sito si aggiorna in automatico in ~30 secondi
